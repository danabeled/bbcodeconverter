[U][B]A guide to this guide:[/B][/U]

~Each weapon has a number next to it. This is important. For a weapon to have that name and appearance, it must contain ALL of the abilities listed next to it.

~If it contains all of the abilities, and you customize it to contain other abilities, it will take the name and form of the HIGHEST numbered weapon on the list. 

For example: 
If a Sword contains the sensor ability, it will be the Hunters Sword, which is #49. If you add Poison Strike, which is the ability on weapon #34, Sidewinder, the Hunter Sword will becoming the Sidewinder because #34 is a higher number than #49.

~A weapon can contain up to FOUR slots for skills and in some weapons, needs to have four slots.
